===========================
Welcome to predictably_core
===========================

``predictably_core``'s goal is to make it easier to build Python packages. It includes
classes the make it easy to reuse `scikit-learn`_, and `sktime`_ style design patterns.
It also includes common tools that make building packages easier. Explore the
rest of the documentation to learn more.
